# pixel7-zygote32 KPM Module

> **作用**：在 Pixel 7 上通过 **Apatch 的 KPM 模块**尝试启用 `zygote64_32`（默认 64 位，兼容 32 位）。  
> **原理参考**：<https://www.jarvisw.com/?p=1587>

---

## ✨ 特性

- 以 **KPM 模块**形式打包，和官方 demo 模块一致的编译、加载流程  
- **选择交叉编译链后即可编译**
